How to Play 2048

Desktop Controls:

  • Use Arrow Keys (↑, ↓, ←, →) to move all tiles in that direction
  • Tiles with the same number merge into one when they collide
  • After each move, a new tile (either 2 or 4) appears in a random empty spot
  • The goal is to create a tile with the number 2048
  • The game ends when there are no more moves available

Mobile Controls:

  • Swipe in any direction to move all tiles that way
  • Tiles with the same number merge into one when they collide
  • After each move, a new tile (either 2 or 4) appears in a random empty spot
  • The goal is to create a tile with the number 2048
  • The game ends when there are no more moves available

Scoring:

  • When two tiles merge, the new tile's value is added to your score
  • Your best score is saved automatically

Good luck and have fun!
